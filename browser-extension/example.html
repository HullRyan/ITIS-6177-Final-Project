Welcome to my page

This is a basic HTML page with some styling, but unfortunately whoever developed this did not think to include Alt attributes for the images (silly me!).

This means that screen readers will not be able to read out the content of the images, which is a problem for people who are blind or have low vision.

Fortunately, if you have the Vision Helper extension installed, so you can use it to add Alt attributes to the images on this page.

This is done automatically when you open the page, it can even OCR (recognize text in images) and add that to the Alt attributes! So screenreaders can now describe the image, and read specific text from it.


Try inspecting the images with the extension off and refreshing the page (no alt attrubutes!), then turn the extension on and refresh the page again (nice).
